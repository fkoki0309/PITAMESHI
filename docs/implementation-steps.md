# 実装ステップ

各ステップは「1機能 or 1画面」単位。UIレイアウト確認 → 機能実装の順で進める。

---

## STEP 1: プロジェクト初期化・環境構築
**ブランチ:** `feature/step1-project-setup`

- [ ] `create-next-app` でNext.js 14 + TypeScript + Tailwind CSS をセットアップ
- [ ] shadcn/ui の初期設定
- [ ] Framer Motion インストール
- [ ] 環境変数ファイル（`.env.local`）のテンプレート作成
- [ ] Vercelプロジェクト作成・GitHubリポジトリ連携
- [ ] Supabaseプロジェクト作成

**確認ポイント:** `npm run dev` でトップページが表示されること

---

## STEP 2: Supabase DB設計・初期スキーマ
**ブランチ:** `feature/step2-supabase-schema`

- [ ] `supabase/schema.sql` を作成（全テーブル・インデックス・RLS）
- [ ] Supabaseコンソールでスキーマを適用
- [ ] 匿名認証を有効化
- [ ] Supabaseクライアント設定（`lib/supabase.ts`）

**確認ポイント:** Supabaseコンソールでテーブルが正しく作成されていること

---

## STEP 3: トップ画面 UI
**ブランチ:** `feature/step3-top-screen-ui`

- [ ] スマホ表示最適化（viewport・フォントサイズ）
- [ ] カラーテーマ設定（オレンジ系）

**確認ポイント:** デザイン・スマホ表示の確認。機能はまだ不要。

## STEP 4: 部屋作成機能
